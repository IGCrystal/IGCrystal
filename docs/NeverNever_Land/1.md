# 概述论

瞳孔失了它的光亮，这个世界，也便分崩离析。黄昏一寸一寸的漫过山野，翠绿的回音留下凝固的琥珀。曾是倒映星空的湖面，此刻已被干涸封锁。显现的是锈迹斑斑的暮色，云朵化作窗帘将一切幕遮。
天地黯然失色，深渊自视网膜深处四处散播。起初，从细小的裂纹中萌芽，仿佛枯枝生长于冰面上的影子。直到某个心跳脱轨的瞬间，传来的玻璃穹顶的坍塌声震耳欲聋。其碎片化作月光的齑粉，坠入永夜的海沟。这样就可以逃过一劫了吧？其实不然，饥渴的漂浮磷火正啃食着记忆的碎片，使得一望无际的荒野浮现出一个个生动的意象——别在母亲鬓角的栀子花生机盎然，少年手中的蓝风筝在天空中展翅翱翔，背后的教堂与天空相得益彰。
风，逐渐加大了它的速度，触觉开始变得锋利起来，只见得皮肤上犁出细密的血珠。脑海的药瓶随着夜色发酵，其苦味膨胀成带刺的藤蔓，沿着脊椎攀爬，连接上那虚弱的心脏。潮水数落着没落了韵律的心跳，留下了被光拒绝的贝壳。
迷茫穿透了这21个岁月，虚无已经灌满了整个身躯。喉间开出透明的菌丝，它们迅速地穿透了天花板，在虚无中结出无助的茧房。黑暗包围了天幕，寂静吞没了整个世界。


何时才能获得救赎?


救赎？救赎的轮廓也变得模糊不清，如同海市蜃楼般虚幻缥缈。它似乎存在于遥不可及的彼岸，又似乎只是一个永远无法触及的幻影。在无尽的黑暗中，救赎的光芒也变得黯淡无光，最终融入了永恒的夜色之中。
黑暗继续蔓延，寂静愈发深沉。世界分崩离析，一切都归于虚无。
